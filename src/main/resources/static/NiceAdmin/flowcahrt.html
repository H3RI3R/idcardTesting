System Design and Activity Diagram

System Design Overview
Admin Module
  • Admin Dashboard: Central control panel for admin tasks and monitoring.
  • Create Table: Functionality to create new game tables.
  • Delete Table: Ability to remove existing game tables.
  • View Bets: Monitoring and reviewing bets placed.
  • Set Winner: Designate winners based on betting outcomes.
  • Referral Commission: Set and manage referral commission rates.
  • Block User: Block or suspend user accounts.
  • Payout Requests: Handle and approve or reject user payout requests.
  • Manage Queries: Oversee user queries and support requests.
  • View Active Users: Monitor currently active users.
  • View Active Tables: Track currently active game tables.
  • System Monitoring: Continuous monitoring of system performance.
User Module
  • User Login: Authentication process for users.
  • User Dashboard: Main interface for users.
  • Join Lobby: Action to join a game lobby.
  • Place Bet: Functionality for placing bets.
  • Pay-In: Recharge user wallet.
  • Request Payout: Request to withdraw funds.
  • Referral System: Manage and track user referrals.
  • Manage Profile: Update profile information.
  • Play Game: Engage in the game.
  • Contact Us: Reach out to support.
  • View Token Balance: Check the balance of tokens.
Additional Features
  • Payment Gateway Integration: Facilitates transactions and payment processing.
  • Token Management System: Handles token issuance and tracking.
  • Notification System: Sends notifications related to updates and game status.
